## Installation

```bash

# crear entorno virtual de python

python -m venv .venv

.venv/Scripts/activate # en windows
. .venv/bin/activate # en linux

# instalar dependecias

pip install -r requirements.txt

```

## Usage

```bash
python manage.py runserver
```
